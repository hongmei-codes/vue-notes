Add vue from CDN

In the head tag add:

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

Vue instance

A vue instance is declared like so:

var vm = new Vue({ // options })

Hello world

Say hello world with vue by adding these lines of code:

In main.js:

new Vue({ el: '#hello', data: { greeting: 'hello world'}})

In index.html:

<div id='hello'>
 <p>{{ greeting }}</p>
</div>

{{ greeting }}

Dynamically access data

Use the double-mastach syntax: {{ variableName }} to access data in vue

e.g. output from vueğŸ‘‡
first name is {{ firstName }}
last name is {{ lastName }}
profession is {{ profession }}

Access method in vue

Use the double-mastach syntax: {{ methodName(param1) }} to use method in vue

e.g. output from vueğŸ‘‡
calling greet() gives {{ greet() }}
calling add(2, 3) gives {{ add(2, 3) }}

Assign vue data to html attributes

Use data binding: v-bind:href="link" to assign vue data as html element attribute. Shortcut for v-bind is: :href="link". Value of href will be rendered and assigned the value of link stored in vue data.

e.g. output from vueğŸ‘‡
Link

Two-way data binding

Use two way data binding: v-model:"name" to update vue data. As user input gets updated, vue data gets updated too.

e.g. effect of two way data bindingğŸ‘‡
vue data: {{ name }}
